#CSS WAHF Autobot#

The CSS WAHF Autobot sends a text to the scholar asking them to send back their quiz grades and how they are doing each semester then will auto-complete their WAHF for them.

##How to run webFunctions##

run `node webFunctions [id]` where id is your place in the `"scholar":[]` array. Right now their are only Marcelo(id:0) and Chris(id:1)
